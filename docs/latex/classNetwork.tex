Creates random connections and assigns them to the Neurons. 

Fills a fixed-\/size table with uniformly distributed values between {\ttfamily min} and {\ttfamily max}. The generated numbers represent the indices of the sources of the neuron at index {\ttfamily idx}. Assigns the neuron at index {\ttfamily idx} as target of the generated sources.
